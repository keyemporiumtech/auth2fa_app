Il componente ChartComponent permette la visualizzazione di grafici a barre verticali. I parametri necessari sono:
  • data: ChartElementInterface[] | ChartElementGroupInterface[] gli elementi da visualizzare in grafico
  • legendTitle, xAxisLabel e yAxisLabel per le label da visualizzare
  • height l'altezza del container (default 400px)
  • options.ColorScheme: ChartColorSchemaInterface definisce i colori da usare nelle barre
  • options.ColorSchemeName: string definisce i colori da usare secondo il set a disposizione dal component 
    


      ```typescript
          this.dataVB = [
            {
              name: 'Pen',
              value: 30,
            },
            {
              name: 'Book',
              value: 50,
            },
            {
              name: 'Table',
              value: 350,
            },
          ];

          this.optVB = {};

          this.dataVBColor = [
            {
              name: 'Pen',
              value: 30,
              color: '#000000',
            },
            {
              name: 'Book',
              value: 50,
              color: '#777777',
            },
            {
              name: 'Table',
              value: 350,
              color: '#FF5733',
            },
          ];

          this.optVBScheme = {};
          this.optVBScheme.colorSchemeName = 'nightLights';


        ```

        ```html
          <ddc-init-chart
            [data]="dataVB"
            [options]="optVB"
            [typeChart]="chartType.VERTICAL_BAR"
            [xAxisLabel]="'Products'"
            [legendTitle]="'Product Sale Chart'"
            [yAxisLabel]="'Sale'"
          ></ddc-init-chart>

          <ddc-init-chart
            [data]="dataVBColor"
            [options]="optVB"
            [typeChart]="chartType.VERTICAL_BAR"
            [xAxisLabel]="'Products'"
            [legendTitle]="'Product Sale Chart'"
            [yAxisLabel]="'Sale'"
          ></ddc-init-chart>

          <ddc-init-chart
            [data]="dataVB"
            [options]="optVBScheme"
            [typeChart]="chartType.VERTICAL_BAR"
            [xAxisLabel]="'Products'"
            [legendTitle]="'Product Sale Chart'"
            [yAxisLabel]="'Sale'"
          ></ddc-init-chart>

        ```

ChartComponent - Vertical and Horizontal
Il componente ChartComponent permette la visualizzazione di grafici a barre verticali.
I tipi VERTICAL_BAR e HORIZONTAL_BAR sono usati per elementi ChartElementInterface
I tipi VERTICAL_BAR_GROUPED, HORIZONTAL_BAR_GROUPED, VERTICAL_BAR_STACKED, HORIZONTAL_BAR_STACKE, VERTICAL_BAR_NORMALIZED e HORIZONTAL_BAR_NORMALIZED sono usati per elementi ChartElementGroupInterface

I parametri da usare sono
  • legendTitle, xAxisLabel e yAxisLabel per le label da visualizzare

      ```typescript
        this.dataVB = [

            value: 30,

            value: 50,
          },
          {
            name: 'Table',
            value: 350,
          },
        ];

        this.dataVBGroup = [
          {
            name: 'Gruop1',
            series: [
              {
                name: 'Pen',
                value: 30,
              },
              {
                name: 'Book',
                value: 50,
              },
              {
                name: 'Table',
                value: 350,
              },
            ],
          },
          {
            name: 'Gruop2',
            series: [
              {
                name: 'Sky',
                value: 300,
              },
              {
                name: 'Dazn',
                value: 150,
              },
            ],
          },
        ];
      }
      ```

ChartComponent - PIE
Il componente ChartComponent permette la visualizzazione di grafici a torta.
I tipi sono PIE, PIE_ADVANCED e PIE_GRID

I parametri da usare per PIE sono
  • legendTitle per la label da visualizzare in leggenda
  • showLabels: booleanindica se mostrare le label sui pezzi
  • explodeSlices: booleanse a true ridimensiona i pezzi di torta in base al valore
  • doughnut: booleanse a true mostra la torta a ciambella
mentre per PIE_ADVANCED e PIE_GRID non sono necessari 
    



      ```typescript
        this.dataVB = [
          {
            name: 'Pen',
            value: 30,
          },
          {
            name: 'Book',
            value: 50,
          },
          {
            name: 'Table',
            value: 350,
          },
        ];
      ```

      ```html
        <ddc-init-chart
          [data]="dataVB"
          [options]="optVB"
          [typeChart]="chartType.PIE"
          [legendTitle]="'Product Sale Chart'"
        ></ddc-init-chart>

      ```
